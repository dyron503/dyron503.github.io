关于生活 故事是生活的比喻，而不是生活的内容或直接反映。 好的故事即使离开视觉效果也依然能抓住观众。 关于手法 手法是故事本身的增益，而非作品着重点。 好的手法可以将好的故事推向极致。 应该和不应该做的事 挖掘和表现生活的意味 在中立情节中通过艺术手法来表达深层含义 在应该的时候，结束故事 罗列琐碎的生活细节，忽视了“大写的真实” 堆砌动作和刺激吸睛的情节起伏 停留在表面对大事件细节的陈词滥调 故事的结构 故事的单位。 关于结构 节拍构成场景，场景构成序列，序列构成幕，幕组合成故事。 结构 结构是对人物生活中一系列事件的选择。用于激发特定情感并表达某种人生观。承载着激发读者共鸣和表达作者观点的作用。 事件 任务生活情境中富有意味的变化。通过一种价值来表达和经历。 故事价值是人类经验的普遍特征。它在正反两面间能够相互转化。转化的过程往往是推动变化的手段或内容。 以依附于故事事件的样态出现。 二者的关系：故事事件是用一种价值来表达和经历的，并通过冲突来完成。 场景 在某一段连续时空中，通过冲突表现出来的一种动作。一般表现为一个确定的时间和空间的统一体。 动作应当使人物生活中负荷着价值的情境发生转折。 理想的场景就是故事事件。 非事件：场景中有一些活动，但价值并未变化-这个场景仅用于说明背景或人物心理的相关信息。 应该和不应该做的事 让每一个事件都具有目的。 平衡事件引发观众共鸣和表达作者思想的比重。 让变化发生在人物身上；或/且由冲突而生。 使动作对观者来说清晰可感。 让每一个场景成为真正的事件。换句话说，使场景包含转折。时时审视场景，努力做到让每一个场景中的价值突出可感，性质清晰，并确保场景的发生使价值的性质发生了些许变化。 将解说编制在故事的各处。 在非绝对必要的时候使用非事件。 编织事件时用纯粹的偶然事件（巧合）来推动。 节拍 动作/反应中一种行为的交替。这些变化的行为通过一个又一个节拍构筑了场景的转折。节拍构筑场景。 在两个节拍切换之时，人物行为有所区别，动作/反应发生明显变化。 序列 节拍构筑场景，场景组成序列。序列是一系列场景（2-5个），每一个场景的冲击力呈现递增趋势。 场景之间有价值层面上的转折。 幕 表现人物生活中负荷价值情景里一个重大逆转的动态单位。它是一系列序列的组合，以一个高潮场景作为顶点，导致了价值的重大转折。 冲击力强于所有前置的序列或场景。它一般体现在人物的内心生活，人际关系或是世事时运中。 故事 作品的主事件，是最大的情节单位。 首尾的价值和/或情境应发生巨大变化。 这一变化-即最后一幕的高潮-是不可逆转的。 故事高潮：一系列幕的组合渐次构成的最后的幕高潮。引发出绝对而不可逆转的变化。 场景/序列/幕的变化是可以逆转的。 应该和不应该做的事 写作场景时梳理和分析其中每个节拍的作用，以此简化和精炼情节。 给序列定下题目和注释，明确它的作用。（如：“故事目的”。） 让每一个场景在自己的层面上发生转折——为每一个场景赋予自己的价值（变化）。 在开始写作时构思故事开头和末尾价值负荷的变化，并在中间的过程中不断提醒自己。 故事高潮应当符合世界中一切人或事变动的逻辑。 让对白或描写服务于行为和动作的转折或情节设置的变化。 让人物对白或最小行为的目的清晰表露在外面而忽视了逻辑。 使一个序列过多的只为情节发展/人物心理变动/人物关系/世界变动等服务。 在“场景间”转换地点而不改变价值。（这样只会创造出结构混乱的单个场景）

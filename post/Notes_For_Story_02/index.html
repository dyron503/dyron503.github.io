《故事》的笔记·贰

· 2021-04-18 · # 笔记 # 写作

结构与背景

打破常规

许多情节都会陷入范式化的泥潭，让读者提前猜到故事的发展；这样的故事往往是失败的。
主要原因在于作者不了解他（她/其它）故事中的世界；这份不了解使得作者被迫从相似的作品而非现实汲取创作灵感。

故事的背景

概论

故事的背景包括时代、期限、地点和冲突层面。

时代

时代是故事在时间中的位置。

很有必要给出明确的设定：现在、过去或是未来；完全的幻想背景则需要以某个现实时代作为参照。

期限

期限是故事在时间中的长度。

一般来说，故事时间长于银幕时间。但通过交叉剪辑、叠印、重复和慢镜头等方式可以在序列中创造长于故事时间的银幕时间，即放慢速度。

地点

地点是故事在空间中的位置。

冲突层面

冲突层面是故事在人类斗争的层级体系中的位置。

可以在不同层面上加以切分：内化/外化、人际之间/社会机构/环境力量，等等。

结构和背景的关系

背景界定了结构的可能性。

故事必须遵守自身内在的或然性[1]法则。因此，在选择事件时，作家被局限于所创造世界内的可能性和或然性。也就是说，不同的背景决定了不同的因果原理；在设定内容确定下来后就不可单独更改。

背景会约束故事设计。

世界的设计应使作者能够了解其中任何可能的细节为宜。过为简略或琐碎的设计都会制约而不是促进故事的写作。请注意，“概念设计”与“世界观”是不同的。

建构世界的方法

  • 记忆研究：从自己过去的经历获取灵感。只有以文字形式记录下来的记忆才是可用的。
  • 想象研究：想想自己处于角色的位置，经历那些他（她/其它）会经历的事情，并由此找到世界中不同因素的关联。
  • 事实研究：查阅资料，询问他人，从现实中他者的经历中获取灵感；通过获取新的自然/人文知识扩充自己的想象空间。

创作选择

写作时事件的量乃是最终故事中事件的量的数倍。创作是取舍的过程中做出的创造性选择。

应该和不应该做的事

  • 以自己想法而非其他作品作为基点展开背景创作。
  • 设定具体的背景；也就是说，明确背景的四个层面。
  • 在背景设计中，突出冲突的地位。
  • 列出并以此明确由世界观决定的应有/不应有的要素。
  • 反复诘问，直到明确自己了解背景细节后再开始动笔写作。
  • 在创作中穿插对相关知识的研究，不断扩宽写作的空间。
  • 列举事件可能的背景，在长长的清单之中做选择。检查的流程：
    • 场景是否适合人物？
    • 场景是否适合世界设定？
    • 场景设计是否新鲜？
    • 如果不得不使用一个常见于其他作品中的场景，那么在细化带入的方式上，重复之前的整套流程。可能还需要对现实中类似的情况进行研究。
  • 通过相似背景的作品（即使媒介完全不同）来大量填充素材。
  • 试图通过模糊背景的方式来逃脱世界观对情节设计的限制。
  • 在背景设定中设计过多无用的细节。
  • 过长时间地沉浸于研究而不反馈到创作之中。
  • 对自己的每个想法都尽力保留而非大胆舍弃绝大部分。

结构与类型

电影类型

一个（尤其是商业）故事许有多种类型，从爱情到体育再到科幻。不同类型相互影响，相互融合。创作之前给自己的故事划分类型很有意义。常见的电影类型有：

  • 爱情故事
  • 恐怖片：
    • 离奇恐怖：恐怖现象可进行理性诠释
    • 超自然恐怖：恐怖力量来自不可解释之物
    • 超级离奇恐怖：到底是什么，你猜~
  • 现代史诗：个人对抗国家
  • 西部片
  • 成长故事
  • 救赎情节：主人公内心由坏到好的变化
  • 惩罚情节：好人变坏并受到惩罚
  • 考验情节：保持坚强意志抵抗诱惑
  • 教育情节：主人公人生观、他人观或自我观由负面到正面的深刻变化
  • 幻灭情节：世界观由正面到负面的深刻变化
  • 喜剧：
    • 恶搞喜剧
    • 讽刺喜剧
    • 情景喜剧
    • 浪漫喜剧
    • 荒诞喜剧
    • 闹剧
    • 黑色喜剧

讽刺力度逐渐上升。

  • 犯罪
    • 神秘谋杀：以侦探大师为视点
    • 罪行：以犯罪大师为视点
    • 侦探：警察的视点
    • 黑帮：匪徒的视点
    • 惊悚或复仇故事：被害人的视点
    • 法庭：律师的视点
    • 报纸：记者的视点
    • 谍战：间谍的视点
    • 监狱戏：囚犯的视点
    • 黑色电影：集罪犯、侦探和荡妇受害者为一身的主人公的视点——？？？
  • 社会剧：展现社会问题并通过故事体现疗救办法，包括：
    • 家庭剧：家庭内部问题
    • 女性电影：展现事业对家庭（等在某些时候往往多发于女性，而非是只能或应该发生在女性身上——笔者注）的困境
    • 政治剧
    • 生态剧
    • 医药剧：与疾病斗争
    • 精神分析剧：与精神疾病斗争
  • 动作和探险：往往以其他类型作为火爆场面的动机。
    • 高端探险：包含命运、狂妄或是精神等概念
    • 灾难/生存电影：对抗力量为大自然
  • 历史剧：往往借古讽今，用古代人事来反映现代实际。
  • 传记：以过去的某人为焦点，找出主体的生活意义并将其树立为这一类人的代表。
    • 自传：难以写好。
  • 纪实剧：很像历史剧，但集中于最近的事件。
  • 嘲讽纪录片：完全虚构的以纪实形式出现的作品，用来讽刺现实中的人或事。

以下类型往往与之前提到的具体类型结合出现。

  • 音乐片：在唱跳的同时演绎故事。
  • 科幻：以假想的未来为背景。
  • 体育类型：以体育承载人物变化。
  • 幻想：架空的世界。
  • 动画：人工创作的卡通画面。[2]

结构和类型的关系

不同类型的故事各有其常规范式。

类型常规

界定各个类型以及次类型的具体背景、角色、事件和价值。

许多类型的常规是高度固定的。但在更多时候，它们有很强的可塑性。

精通类型

为了预知观众的预期，创作类型作品之前，作者应该对类型有充分了解。进行类型研究的方法如下：

  • 列出属于该类型的作品清单，既包括成功的作品，也包括失败的；
  • 准备作品，（如果有条件的话）找到它们的剧本；
  • 拉片，对剧本结合画面进行分析；分析的对象是背景、角色、世界和价值等；
  • 从上到下观察你的分析；
  • 思考此类型故事的作用以及时间、地点、人物和动作的常规。

创作限制

确定统领作品的类型常规会给创作加上人为限制，但这有利于激发作者的灵感。
当然，在常规的范围内避免陈词滥调是不小的挑战。在范式情节上推陈出新将为作品赋予最大的亮点之一。

混合类型和再造类型

类型是发展的。它随着时代的变化相应改变，
反映时代的变化，回答时代的问题。

应该和不应该做的事

  • 自我发现，及时确定作品类型。
  • 开始创作前确保自己对类型的特征有较深入的了解。
  • 让作品外延部分（如衍生品）的气质与类型气质相符合。
  • 通过精通类型，打破常规的具体表达方式。
  • 了解当下，推测社会未来的方向，以此对自己的作品类型在通用类型上做些创新改造。
  • 选择自己能够认同的类型进行创作。这有助于长期坚持。
  • 使用落后的剧情范式（如子女对包办婚姻言听计从）
  • 在利用类型为观众营造出某种特定预期的情况下省略或滥用常规。

  1. 前提与结论间不具有蕴涵关系的推理，即从真前提不能必然而只能或然地推出真结论的推理。典型的例子：归纳法。 ↩︎

  2. 由于此书写作年代的局限，这里提到的“动画”远不如现在我们看到的那般丰富和广泛，因此只是作为类型存在；这与如今的情况已有些不同。——笔者注 ↩︎
